Branchly Privacy Policy

Last updated: April 1, 2026

Overview

Branchly is a Chrome extension that helps you branch off ChatGPT and Gemini conversations into new contextual threads. Your privacy is important to us, and Branchly is designed to be fully local and private by default.

Data Collection

Branchly does not collect, transmit, or share any user data. Specifically:

  • No personal information is collected
  • No analytics or tracking tools are used
  • No data is sent to any server or third party
  • No cookies are set by the extension

Data Storage

All data created by Branchly (your branch tree, conversation context, and preferences) is stored locally in your browser using IndexedDB and Chrome's storage API. This data never leaves your device.

Website Access

Branchly only operates on the following websites:

  • chatgpt.com
  • chat.openai.com
  • gemini.google.com

The extension reads conversation content on these sites solely to provide context when creating a new branch. It does not access, read, or interact with any other websites.

Permissions

Branchly requests the minimum permissions necessary to function:

  • sidePanel — to display the branch tree navigation panel
  • activeTab & scripting — to read selected text and automate creating new chat threads
  • storage — to persist extension state across browser sessions
  • tabs — to navigate to branch conversations when clicked

Changes

If this privacy policy changes, the updated version will be posted here with a new "last updated" date.

Contact

If you have questions about this privacy policy, please open an issue on GitHub.
